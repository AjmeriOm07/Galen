# 🏥 Galen - Disease Prediction System

A symptom-based disease predictor using **Natural Language Processing (NLP)** and **Random Forest** classification. Built as a college project for the Data Mining course.

## 📌 Overview

Galen takes user symptom descriptions (e.g., *"I have fever, headache, and body pain"*), extracts symptom keywords using NLP, and predicts the most likely disease(s) using a **Random Forest classifier** trained on a dataset of 42 diseases and 131 symptoms.

## ✨ Features

- 🗣️ **Natural Language Processing** for symptom extraction from free text
- 🌲 **Random Forest classifier** for multi-class disease prediction
- 📊 **Probability-based** prediction output with confidence percentages
- 💬 **Simple chat interface** for symptom input
- 📋 **Report generation** with predicted diseases

## 📂 Dataset

- **Source:** Kaggle
- **Diseases:** 42
- **Symptoms:** 131
- **Format:** CSV

🔗 [Link to Dataset][(https://www.kaggle.com/datasets/itachi9604/disease-symptom-description-dataset)
